甦醒航班 · 系統怎麼串在一起

Vibe Coding 入門 · 以 Sleep Airline 為例

基本觀念

程式 = 文字檔

  • HTML：頁面結構
  • CSS：外觀樣式
  • JS / TS：互動與邏輯

前端 vs 後端

  • 前端：在瀏覽器執行，負責畫面
  • 後端：在 Vercel 執行，負責資料與 AI

JSON

  • 前後端溝通用的資料格式
  • 登入、起飛時傳遞姓名與小隊

Notion = 資料庫

  • 航班紀錄存在 Notion 表格
  • 起飛新增一列，降落更新同一列

AI 的角色

  • 寫程式：Cursor、Claude、Codex
  • 產品內 AI：OpenAI 生成機長廣播

GitHub 與 Vercel

  • 下載 ZIP → 本機改完 → 推到自己 GitHub
  • Vercel Import 學員 repo，自動發布網站
